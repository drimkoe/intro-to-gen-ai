# Introduction to Generative AI - Lab Instructions

Welcome to the lab instructions repository for the Introduction to Generative AI course!

This repository contains the markdown instructions for each lab session. Because some virtual machine environments restrict copy-and-pasting, you can use these files directly or view them here on GitHub to easily copy code snippets and follow along with the exercises.

## How to Use

*   **View Online**: You can simply click on the links above to read the instructions directly on GitHub.
*   **Clone Locally**: If you want to have the instructions available in your local environment, you can clone this repository using:
    ```bash
    git clone https://github.com/drimkoe/intro-to-gen-ai.git
    ```

## Note on VS Code and Jupyter Notebooks

If you are using Visual Studio Code to open the `.ipynb` notebooks for the first time, you will need to install the **Python** and **Jupyter** extensions when you select a kernel. VS Code should prompt you to do this automatically.

Happy learning!
